<h3>Installation</h3>

 `$ git clone https://github.com/thebozturk/rabbitmq-tutorial.git` 
 
 `$ cd  rabbitmq-tutorial `
 
 `$ npm install `
 <br>

 <h2>Usage</h2>
 
- Run ' node publisher.js "queueName" ' in a terminal and run ' node consumer.js "queueName" ' in a other terminal.
